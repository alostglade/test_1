仓库（repository）
用来存放项目代码：repository
多个开源项目有多个仓库

star（收藏）
收藏项目，方便下次查看

fork（复制收藏项目）
复制成一模一样的仓库，并且是独立存在的

pull request
原项目是张三的，李四fork了张三的仓库，然后想做一些改动，就写代码然后向张三发
起pull_request并且如果张三感觉不错，他就会把代码合并到原仓库中

watch（关注）
你watch了一个项目，你就可以第一时间收到这个项目的目前更新

一个git库对应一个开源项目，我们通过git管理git库

创建仓库
一般来说仓库名对应项目名
description描述项目
readme.txt用来详细描述项目

编辑文件算一次提交
